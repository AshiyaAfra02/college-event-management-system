# College Event Management System (CEMS)

A Java-based desktop application developed using **Java Swing, NetBeans, JDBC, and MySQL** to streamline college event planning, student registrations, and administrative event management.

## Features

* User login and role-based access
* Event creation and management
* Student event registration
* Organizer and admin management panels
* Venue and notification management
* MySQL database integration for storing event and registration data

## Tech Stack

* Java
* Java Swing
* NetBeans IDE
* JDBC
* MySQL

## Project Structure

* `src/cems/` – Java source files for UI, models, and DAO classes
* `nbproject/` – NetBeans project configuration files
* `manifest.mf` – Project manifest file

## Key Modules

* User Management
* Event Management
* Registration Management
* Venue Management
* Notification Management

## How to Run

1. Open the project in **Apache NetBeans**.
2. Configure the MySQL database in `DBconnection.java`.
3. Create the required database tables in MySQL.
4. Run the project from NetBeans.
